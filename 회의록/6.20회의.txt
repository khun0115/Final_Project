6.20 회의

회의안건 : 최종모델 구축을 위한 사전작업

	1. 수리비용의 경우, JSON파일의 이상치가 많고 NULL값도 다수 존재해 사용하기 어렵다고 판단
	2. 수리비용에 관한 데이터를 추가수집
	3. 파손위치 CNN모델의 경우 범퍼, 문, 휀더의 이미지가 구분이 어렵고, 가격의 편차가 크지 않으므로 통합
	4. ML 모델 구축시 target 
		BEFORE : 수리방법중 하나를 도출해서 가격제시
		AFTER : 수리시간을 도출한 후 수집한 DB의 부품 및 수리가격 적용
		* DB : 차량 부품의 가격, 수리가격, 도장가격등 

	5. 진행상황	
		DL 모델: 학습중 
		DB 구축 : 데이터 수집중
		통계 검증 : 추가 데이터 수집후 통합해서 진행예정

	6. 할일 : 
		YOLO모델 학습 - 진성
		파손모델 이미지파일 축소 - 현욱, 도현
		DB 데이터 수집 - 영익, 이경
		차량 부위중 범퍼, 휀더, 문 앞뒤 구분없이 통합 - 도현, 현욱
		차량 부위중 문(DOOR) 데이터 추가 수집 - 이경
		가격, 색상, 부위  DB 구축 - 영익, 은경, 현욱
